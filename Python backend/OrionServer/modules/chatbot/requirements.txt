# =============================================================================
# Core System Architecture
# =============================================================================
# Hybrid Storage Tiers:
# - Redis (0-3h): Hot cache for active conversations
# - Qdrant (3h-7d): Vector search for recent chats + semantic retrieval
# - Firebase (7d+): Long-term persistent storage
#
# LLM Integration:
# - DeepInfra API (Meta-Llama-3.1-8B-Instruct-Turbo)
# - Vision processing for image attachments
# - Streaming responses via Server-Sent Events
#
# Progressive Memory Management:
# - Smart conversation summarization (every 8 messages)
# - Zero context loss with progressive summary chains
# - Summary nodes seamlessly stored across all storage tiers
#
# Automated Migration System:
# - Intelligent chat migration based on access patterns
# - Redis → Qdrant (12+ hours inactive)
# - Qdrant → Firebase (7+ days inactive)
# - Automatic cleanup with safety verification
# - Access time tracking across all storage tiers
#
# Chat ID Format: {user_id}::CHAT::{video_id}
# Message Sequencing: Redis INCR + deterministic UUIDs
# =============================================================================

# - Zero context gaps: complete conversation history preserved via summary chains
